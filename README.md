# sistema-estoque

   Aprendendo CRUD.
   
Projeto desenvolvido com Flask e PostgreSQL para gerenciamento de produtos.

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Edição de produtos
- Exclusão de produtos
- Validação de campos
- Integração com PostgreSQL

## Tecnologias Utilizadas

- Python
- Flask
- PostgreSQL
- HTML5
- CSS3
- Git
